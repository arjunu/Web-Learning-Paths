# Git Cheatsheet

? - indicates optional arguments

## Create/Clone

- Create new local repo<br>
`$ git init <?project-name>`

- Clone existing <br>
`$ git clone <url>` 

- Create from existing code <br>
`$ git init` <br>
`//add files & commit` <br>
`$ git remote add origin <remote>` <br>
`$ git push origin master` <br>
